APIs and Web Storage in HTML5

HTML5 introduced powerful new APIs for web applications, including Web Storage and Geolocation.

Web Storage

Web Storage allows web applications to store data locally on a user's browser.

    // Example of using localStorage
    localStorage.setItem("username", "John Doe");
    let user = localStorage.getItem("username");
    console.log(user);  // Output: John Doe

Geolocation API

The Geolocation API allows web apps to access the user's geographic location.

    navigator.geolocation.getCurrentPosition(function(position) {
        console.log(position.coords.latitude, position.coords.longitude);
    });
